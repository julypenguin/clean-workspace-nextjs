# React Multiple Platforms

> 多商戶架構實踐

## 環境

- `nvm`

- `node ^20.14+`

- `yarn ^1.22+`

  ```sh
  yarn install
  ```

## SSG Demo

```sh
yarn build ssg
yarn start
```

## 啟動說明文件

> 直接閱讀 `document/docs/*.md` 也可

- 啟動文件服務

  ```sh
  yarn dev document
  ```

- 或編譯文件

  ```sh
  yarn build document # 體驗 rspack 的魅力
  yarn build start
  ```
